Multiple allele queries
get

/alleles

This is a common GET path where more than one allele is expected in the response. One of the example is searching ClinGen Allele Registry using HGNC gene symbols. Please note that the query parameters: gene, dbSNP.rs, ClinVar.VariationId, ClinVar.RCV, MyVariantInfo_hg19.id, ExAC.id, and, gnomAD.id are mutually exclusive. You just need to provide one of them for query.
